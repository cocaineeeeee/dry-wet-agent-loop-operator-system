## 1. 形式化设定

**观测模型。** 设计空间 `X`，来源特征空间 `Z`（批次标签、孔位、捕获序等）。每条观测：

```
y_i = f_θ(x_i)  +  a(x_i, z_i; ω)  +  ε_i
```

- `f_θ(x)`：**真值面**，被估参 `θ`（我们真正想要的量 —— 真最优位置 / 真条件对比 / 真效应）。
- `a(x, z; ω)`：**结构化伪影**，随来源 `z` 与讨厌参 `ω` 变化的系统性偏差（批次位移、边缘蒸发、
  温度梯度…）。关键：它依赖 `z`，不是 iid 噪声。
- `ε_i`：零均值噪声，`ε_i ~ N(0, σ²)` iid，两世界共用同一噪声律。

**两个 setting。** 来源特征 `z` 可见（provenance-aware）/ 不可见（聚合盲）。

**估计器类（这是命题的真内核 —— scoping）。**

- **聚合盲估计器** `T`：任何只消费 `{(x_i, y_i)}` 的可测函数 `T: (x,y)-样本 → θ̂`。
  中位数、Huber M-估计、稳健 GP（RCGP）、trimmed-mean **全在此类**——它们对 `z` 一无所知。
- **来源感知估计器** `S`：消费 `{(x_i, y_i, z_i)}` 的可测函数。**硬路由（隔离）与软降权
  （regress-out / tempered 加权）同属此类**——两者都用 `z`。

命题裁的是这两类之间的分界，**不是** 稳健 vs 朴素、也**不是**硬 vs 软（后者见 §2 诚实边界）。

## 2. 主命题（Le Cam 两点法）

### 2.1 混淆对构造：批次效应 vs 真条件效应

两候选 `x ∈ {A, B}`，二值批次 `z ∈ {0, 1}`。取常数 `s > 0`，估计对比 `θ = f(B) − f(A)`。

| | 真值面 `f` | 伪影 `a` | A 孔观测均值 | B 孔观测均值 |
|---|---|---|---|---|
| **世界 0（真效应）** | `f(A)=μ, f(B)=μ+2s` | `a ≡ 0` | `μ` | `μ+2s` |
| **世界 1（批次伪装）** | `f(A)=f(B)=μ+s`（无效应） | `a(z=0)=−s, a(z=1)=+s` | `μ` | `μ+2s` |

布局把 A 钉定在批次 0、B 钉在批次 1（`z = 1[x=B]`，完全混叠——这正是 §3 棋盘格自发产生的）。
于是世界 1 里：A 孔 `= (μ+s) + (−s) = μ`，B 孔 `= (μ+s) + (+s) = μ+2s`。

**两世界的估计对比不同**：`θ₀ = 2s`（B 真优），`θ₁ = 0`（B 无优、纯批次假象）。

### 2.2 聚合盲极小极大风险 = s（下界 + 可达 = 紧）

聚合盲 `T` 只见 `(x, y)`。逐 `x`：世界 0 与世界 1 的 `y | x` 律**逐点相同**
（A→`N(μ,σ²)`、B→`N(μ+2s,σ²)`，两世界一致），噪声律同。故 `(x,y)`-样本的联合律在两世界
**完全相同**：

```
TV( P₀^{(x,y)} ,  P₁^{(x,y)} )  =  0      （对任意样本量 n）
```

**下界（Le Cam 两点法）。** Tsybakov 2009 §2.2 定理 2.2 给：对任意聚合盲估计器 `T`，

```
inf_T  max_{j∈{0,1}}  E_{P_j} | T − θ_j |   ≥   (|θ₀ − θ₁| / 2) · ( 1 − TV )   =   s.
```

**可达（紧）。** 常数中点估计器 `T ≡ (θ₀+θ₁)/2 = s` 在两世界的风险均为 `|s − θ_j| = s`，其
最坏风险精确 `= s`，恰与下界重合。故聚合盲**极小极大风险精确 `= s`**（不只是"≥ s 的下界"）：

```
inf_T  max_{j∈{0,1}}  E_{P_j} | T − θ_j |   =   s      （紧；红队数值 V1：midpoint 估计器 max-world 风险 = 0.3000 = s）
```

因 `TV = 0`，这个 `s` **与 n 无关**——是信息论地板，不是收敛率。直觉：`T` 的采样律在两世界
逐字相同，它无从知道 B 的高读数是真优还是批次；最优只能猜中点 `θ = s`，两世界各偏 `s`，
比随机猜测强不了。**加样本无救**——地板不随 n 降。

**引理 2.2（自适应盲策略同受地板）。** 上界证明写在固定设计估计器上，但 BO 应用是**自适应**的
（下一查询依赖已见历史）。任意聚合盲**策略** `π` 的每步查询规则只依赖两世界**同分布**的
`(x, y)` 历史；由 `TV(P₀^{(x,y)}, P₁^{(x,y)}) = 0` 且噪声律共用，逐步条件律在两世界相同，
归纳即得序贯交互**整条轨迹的联合律在两世界完全相同**，故其上任意末端估计量的
`TV = 0`。于是极小极大地板 `s` 对**任意自适应聚合盲策略**（而非仅固定设计估计器）成立——
定理从"估计器"升到"策略"，正是 BO 应用所需的强度。

### 2.3 来源感知可分辨（→ 0）

要 `S` 能分辨，布局须提供**跨来源锚点**：比例 `ρ>0` 的 A 孔置于批次 1、`ρ` 的 B 孔置于批次 0
（OS 布局的跨批次副本 / 哨兵，M6 `cross_batch=True` addition remedy）。则：

- 世界 0（`a≡0`）：批次 1 里的 A 孔仍读 `μ`。
- 世界 1（批次伪影）：批次 1 里的 A 孔读 `μ+s+s = μ+2s`。

来源感知 `S` 见 `(A, z=1)` 这组：两世界均值差 `2s`，是纯批次信号。差分-差分 / regress-out `z`
的估计器风险 `≤ O(σ / √(n·ρ))  →  0`（收敛条件 `n·ρ → ∞`；红队数值 V2：`SE·√(n·ρ)` 跨 n 恒定）。**注意该估计器用尽所有孔——包括被
硬路由会隔离掉的批次 1 的 A 孔——它天然是软方法**。

### 2.4 诚实边界（本命题证到哪、不证到哪）

1. **证的是「provenance 必要」**：聚合盲有常数地板 `s`（§2.2，任意 n），来源感知击穿之
   （§2.3）。这把「稳健统计不够」升到定理级。
2. **不证「硬路由必要」**：§2.3 的可辨识估计器是**软的**（regress-out / 加权，用全部锚点孔）。
   硬隔离把批次 1 的 A 孔 QUARANTINE 掉，**丢掉的正是辨识伪影所需的锚点信息**→数据饥饿。
   这与 resweep 实证一致（硬隔离弱档输给软方法，§3.2）。故论文措辞应为「路由是利用设计侧
   辨识信息的**一种充分机制**」，**不写「唯一/必要」**（PAPER_OUTLINE §3.3T 纪律）。
3. **地板对近似混叠的退化**：`ρ>0` 时聚合盲 `TV ≈ ρ`，地板降为 `s·(1−ρ)`（线性退化）。
   小 `ρ` 下聚合盲仍基本卡在 `s`，来源感知已经赢——差距不是 `ρ=0` 的假象（见 §5 攻击点 1）。

## 3. 与实证的桥

### 3.1 混淆对 ↔ S2.batch_shift 场景

模拟器批次分组是**棋盘格** `solution_batch = (row + col) mod n_batches`
（ARCHITECTURE.md `artifacts.py`、M6_DESIGN §批次对齐）。分层随机化把候选副本沿区组铺开时，
候选身份与棋盘格奇偶**天然对齐**——§2.1 的「A 钉批次 0 / B 钉批次 1」不是人造，而是布局几何的
自发后果。**M6 finding 作为该混淆在真系统里自发出现的证据**：`attribution._board_frame` 重建
每孔批次时若用 capture 序 `idx%n` 而非棋盘格 `(row+col)%n`，会把观测错分出真批次组、稀释真
批次效应、令 `t_batch` 失真——缝隙守卫测试 `test_board_frame_batch_matches_simulator_labels`
钉死此对齐。**几何混叠（批次 × 分层奇偶）是归因误差的系统性来源**——正是本命题的混淆对在
生产代码里现身，且需专门守门才不塌。

### 3.2 resweep 数字：命题解释什么 / 不解释什么

- **解释（命题的正面）**：极显著的可信性指标 —— 假最优拒斥 os 0.20 vs naive 1.00
  （精确置换 p≈3.05e-5，溯源 headline_stats.json）、训练集污染率 os 0.004 vs naive 0.146（p<1e-4）。naive 是聚合盲，追世界 1 的
  `μ+2s` 假最优（批次伪影）；os 见 `z` 把它路由出去。这正是 §2.2 地板的实证投影：盲臂无从
  分辨真优与批次假象。
- **不解释（命题诚实不覆盖的）**：regret 主口径 os vs naive **p=0.0668 不显著（S0.demo 单场景口径，stats_tests.csv）**，且弱幅度档硬隔离
  的 os regret 反高于 naive。命题**预测**了这一点：它只证 provenance 必要、**不证硬路由必要**；
  硬隔离丢锚点→数据饥饿（§2.4.2），软方法（os-soft，用 §2.3 的全锚点估计器）才是命题指向的
  achievability。resweep 的「硬<软」不是命题的反例，是命题诚实边界的实证确认。

## 4. 推论与界限（引理化，不展开）

- **P2 软硬相变（remark，引 RCGP/tempered 锚点）**：纯 MSE 下最优权 `w*(s) ∝ σ²/(σ² + n_S·b̂²)`
  恒平滑 ∈(0,1)，永不精确取 0；真「硬相变」`b² ≈ τ²` 需外加 0-1 决策损失 / minimax-over-b。
  为 os-soft v2 权函数提供**推导出的形状**（替线性斜坡）。锚点：RCGP `w ∝ 1/(1+(r/c)²)`
  （arXiv:2311.00463）、tempered posterior BO（arXiv:2601.07094）。**不展开证明**。
  **反例侧（须与正面并列，勿只写 w* 形状）**：当 `b² > τ²` 且批次只得**中等 suspicion**（落
  QUARANTINE 软化带、恒正下限 `w_min` 强制纳入一个 `w*→0` 的组）时，**软路由严格劣于硬**——
  合成 GP（`synth_gp_reversal.py`，`b≥0.3` 起 soft−hard 差 +0.024→+0.25）与实测 batch −0.18 /
  batch_dust（`os-soft − os` 正、重尾）**均现**。适用边界声明（软信任非硬路由超集，其优越性以
  suspicion 对偏差秩一致 + `b²≲τ²` 为条件）见 PREM §B.3。
- **P1 污染预算解耦（引理一句）**：路由把污染预算从响应模型解耦——Huber (1964) ε-contamination
  minimax + breakdown point 的重述，经典锚点，不当新定理。
- **P4 截断正态 QC 税上界（引理一句，降格为 sanity floor）**：硬隔离在干净数据上假阳性的效率
  损失有截断正态解析上界（Tallis 1961 / Johnson-Kotz 截断正态矩）。**但此上界不是保费上限 `X`
  的解析支点**：实测（PREM §A.3）它只覆盖 os−naive 保费的 ~5%（隔离通道），主导通道是零伪影下
  的采集畸变（risk_discount），P4 框架不覆盖。故 P4 降格为**不依赖经验分布的 sanity floor**——
  真正的可证伪定义是 `X = max(P4 下垫, 校准集经验 Q0.90)`（PREM §A.3），二者取大，X 由校准集
  数据钉死、非自由预注册。

## 5. 给红队的攻击点自白（证明里最薄的三处）

1. **混淆对的可实现性（真载荷是伪影类的包含关系，非刀刃几何）**：本命题的真载荷假设是
   **「伪影类 ⊇ 效应等幅的批次混淆」**——即对手只要能构造出一个与真效应等幅、沿 `z` 结构化的
   加性偏差，`TV=0` 的混淆对就落在伪影类内，对手构造是**鲁棒**的（不依赖任何精确抵消的巧合）。
   §2.1 表里的 `−s/+s` 只是这个包含关系的一个**代表元**，不是命题赖以成立的刀刃条件。近似混叠
   `ρ>0` 下地板线性退化到 `s·(1−ρ)`（V3 证实），**红队可推大 `ρ`（OS 多加跨批次锚点）压缩盲
   地板——但那是 OS 在干活、非漏洞**。真正要守的边界是「效应等幅的批次混淆是否落在设定的伪影类
   内」，而非某个几何是否刀刃。
2. **估计器类的定义边界**：「聚合盲」定义为只见 `(x,y)` 的可测 `T`。但真实稳健-GP 臂的 `x` 可能
   已含部分位置特征（edge 距离等）而**间接代理 `z`**。若 `x` 已编码部分来源，盲/感知分界模糊，
   命题的力量取决于 `z` 是否**真被从 `(x,y)` 表征中扣留**。须钉死哪些 provenance 特征入 `x`、
   哪些入 `z`——否则「盲」臂偷看了来源，地板不成立。
3. **多点构造的地板 + 估计目标的可归约性（两回事，分开写）**：注意「本两点对的地板 `s` 未证紧」
   这句**对本对不成立**——§2.2 已用中点估计器证得极小极大风险**精确 `= s`**（可达=下界=紧，
   V1 数值 0.3000 佐证）。真正开放的是**另一件事**：更丰富的世界族 / Fano 多点构造能否给出
   **比 `s` 更大的地板**（可能更大、也可能这条两点对已是最坏对之一）——这是"能否加强命题"的
   开问，与"本对是否紧"无关，勿混。第二处开放（真开放）：本证严格成立的估计目标是**单个对比 /
   效应 `θ`**；延伸到连续设计空间上的 **simple-regret（真最优位置）** 需一步归约（把两点埋进
   最优位置的邻域），本文**未完整给出**——论文若把地板直接搬到 regret 头上，是超出证明的声称。

## 6. 新颖性定位（抗理论审稿人）

**技术上零新颖，明写。** 本命题的方法是 Tsybakov 两点法（Le Cam 1973 / Tsybakov 2009
《Introduction to Nonparametric Estimation》§2.2）的**直译**——不声称任何方法学新颖。审稿人若
以"两点法是教科书技术"质疑，回答是：同意，方法本身零新颖，贡献不在方法。

**贡献在两点，都不在方法。**
- (a) 把 **provenance 形式化为设计时可观测的识别性侧信息**：`z`（批次 / 孔位 / 捕获序）不是事后
  诊断量，而是布局阶段就可写入设计矩阵的识别性变量；命题给出"扣留 `z` ⇒ 不可辨识、纳入 `z` ⇒
  可辨识"的定理级分界，把"要记 provenance"从工程习惯升为可证的识别性要求。
- (b) 指出**混淆对在生产棋盘格几何里自发出现且需专门守门**：§3.1 的棋盘格 × 分层奇偶对齐不是
  人造反例，是真布局几何的后果，M6 缝隙守卫（`_board_frame` 对齐测试）就是这道门——缺它归因塌。

**与 Huber-Ronchetti 稳健统计的对照（威胁模型正交、非竞争）。** 稳健统计防的是 **iid 污染**
（重尾 / 离群点，breakdown point 视角）；本命题防的是 **`z`-相关的结构化偏差**（沿来源系统性、
非 iid）。两者是**正交的威胁模型**：稳健估计器（中位数 / Huber / RCGP / trimmed）全落在聚合盲
类里、同受 `s` 地板（V1 四估计器 max-world 风险全 = 2s），故本命题**不与稳健统计竞争**、而是刻画
它够不到的一个正交轴。

**最近理论亲缘（引 arXiv:2601.11924）。** 2601.11924 在 bandit 设定给出**验证预算的可学性相变**；
我们给出其在 provenance / BO 估计设定下的**辨识性对应**，方法为 Le Cam 两点法。二者共享"额外侧
信息（验证预算 / provenance）决定可学性"的骨架，威胁模型与技术路径各异，非重复。
